Admission Policy

Clat Admission policy is designed to be user friendly and easy to implement .The details are given below


Admission

Admissions to all the course except those explicitly specified will be done on the basis of the eligibility criteria defined in the course introduction page .


Please ensure that you are eligible for the course .


The learners will need to apply for admission as per the process defined from time to time .


Personal Information - Learner Integrity

Learners personal Information - Clat being an online program and follows Honour Code that works on trust .

Based on this Honour code QCI will take all the information shared by the Learner on its face value .

However if at any time during the course and after the course as well if it is discovered that false information has been provided by the learner then QCI holds the right to cancel the certificate and blacklist the candidate for any future programs of QCI and inform appropriate authorities QCI deems fit .

Honor Code

All participants in the Clat course must agree to the following code of conduct:
• They will maintain only one account.
• They will complete all (Quiz, Mid-term & Final) exams with their own work.
• They will not submit their work to any other person.
• They will not engage in any activity that will dishonestly improve their result or hurt the results of the others.
• They will not post answers to problems that are being used to assess participant performance.

Note: This Honor code will be review on timely manner and any changes to this code will be effective immediately. So be sure to this page periodically to ensure with the most current version of Honor Code.
